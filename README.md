# Raptor Starter

This repository is the starter template for **Raptor**, a Swift framework for static sites and server-side rendering.

---

## Getting Started

Create a new site using the Raptor command-line tool:

```bash
raptor new MySite
cd MySite
open Package.swift
```

Build and preview the site:

```bash
raptor build
raptor run --preview
```
